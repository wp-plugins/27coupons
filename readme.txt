== Description ==

This plugin will create a widget which will display latest discount coupons and deals of Indian e-commerce websites from 27coupons.com. 
You can customize the plugin to change the title and number of deals to be displayed.
For further customizations of this plugin, please contact us at http://www.27coupons.com/contact-us/

== Installation ==

1. Download the plugin zip file (27coupons.zip) from http://cdn.diffion.com/widgets/27coupons.zip
2. Unzip and Upload contents of folder '27coupons' to the '/wp-content/plugins/' directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Go to 'Appearance-->Widgets' and add the widget 'Latest Discount Coupons and Deals' to the sidebar.
5. You can change the title and number of deals to be displayed.
